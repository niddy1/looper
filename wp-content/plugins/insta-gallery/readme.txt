Instagram Gallery is an easy way to display your Instagram pictures on the site.

== Description ==

This plugin is an easy and simple way to display your instagram images on the website.
we can display pictures from instagram account as Gallery or as slider. 
this plugin updates new pictures as you upload new-one on Instagram.

**Note:** you have to add your instagram account username/id.
 and profile needs to be publicly visible to show pictures.

**Demo:** live demo is not available yet. you can check 'screenshots' tab for demo.

**How to use:** generate shortcode from plugin panel and 
paste them in the pages/post content or 
use the "echo do_shortcode( '[MyShortcode]' );" php code to add them in PHP file.

= can i add multiple shortcode in a page? =

yes, you can add multiple shortcodes within a page.

= do i have to enter API keys to display pics? =

no, there is no need to enter API keys, just enter username and enjoy.
